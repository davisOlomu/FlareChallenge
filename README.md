Maze Generator App
This project implements a maze generator application built using Windows Forms.

Features:
Generates random mazes of same sizes. 
Visualizes the maze within the application window.

How to Use:
Clone or Download the Repository: Download the project files from GitHub.
Open the Solution: Open the .sln file (or project file) in your preferred IDE (e.g., Visual Studio).
Run the Application: Press F5 or use the "Run" menu option in your IDE to start the application.
Click "Generate" Button: Click the "Generate" button to generate a new maze.
